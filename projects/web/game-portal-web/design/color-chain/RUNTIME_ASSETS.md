# クロマ 実装用ランタイム素材

キャラクターパネル第1段階で使用する背景透過素材。設定画を基準に、同じ正方形キャンバスと足元位置で制作している。

## 配置先

`public/characters/chroma/`

| 状態 | Web表示用 | 保管用 | 用途 |
| --- | --- | --- | --- |
| 通常 | `chroma-idle.webp` | `chroma-idle.png` | 通常待機・軽い上下動 |
| 瞬き | `chroma-blink.webp` | `chroma-blink.png` | 通常画像との短時間切替 |
| 連鎖 | `chroma-chain.webp` | `chroma-chain.png` | マジカルチェイン成立時 |
| 危険 | `chroma-danger.webp` | `chroma-danger.png` | 盤面上部まで積み上がった時 |
| 敗北 | `chroma-defeat.webp` | `chroma-defeat.png` | ゲームオーバー時 |

## 共通仕様

- キャンバス: 1254 x 1254 px
- 背景: 透過
- キャラクター: デフォルメ版クロマ、全身表示
- 基準: 帽子・杖・両足がキャンバス内に収まり、足元を下端寄りの同一基準に配置
- PNGは確認・再編集用、WebPはゲーム内読み込み用
- 通常と瞬きはポーズを揃え、画像差し替え時の位置ずれを抑える

## 制作プロンプトの要点

共通して `chroma-chibi-setting-v1.png` を衣装・人物同一性の基準に使用。単色クロマキー背景上で個別生成し、ローカル処理で透過PNG/WebPへ変換した。

- 通常: 正面寄りの落ち着いた待機姿勢、両目を開く
- 瞬き: 通常の構図・衣装・杖位置を維持し、両目だけ自然に閉じる
- 連鎖: 小さな拳上げ、笑顔、杖の近くに収まる短い三色の魔法鎖
- 危険: 杖を胸元で握り、後ずさり気味の不安な表情
- 敗北: 肩と杖を下げた、静かに落ち込む立ち姿

## 実装時の注意

- `<picture>` ではWebPを優先し、必要ならPNGをフォールバックにする。
- 状態切替でレイアウトが動かないよう、表示枠のアスペクト比を `1 / 1` に固定する。
- `prefers-reduced-motion: reduce` では上下動を停止し、表情切替だけを残す。
- 第1段階ではテスト版 `/test/color-chain-mascot/` だけで使用する。
