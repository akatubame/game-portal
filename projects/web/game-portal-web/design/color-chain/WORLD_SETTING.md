## 舞台

物語の舞台は、色の魔力が鎖によって世界を巡る **彩鎖界カテナリア**。

世界中には「彩鎖（さいさ）」と呼ばれる魔法の鎖が張り巡らされている。彩鎖は誰かを罰する拘束具ではなく、離れた土地・生き物・色の魔力を結び、暴れ出した力を落ち着かせて本来の居場所へ戻すための仕組みである。

かつて「断鎖の嵐」と呼ばれる大きな異変が起こり、各地の彩鎖と、その流れを調整する「結び目」がほつれた。鎖からこぼれた色のかけらは盤面へ降る色ブロックとなり、周囲の魔物も色の力を抱え込みすぎて不安定になっている。

## 基本用語

| 用語 | 意味 |
| --- | --- |
| 彩鎖 | 色の魔力を安全に循環させる、世界規模の魔法の鎖 |
| 結び目 | 地域ごとに彩鎖の流れを調整する要所。将来のステージ拠点に相当する |
| 断鎖の嵐 | 各地の彩鎖がほつれた過去の異変。原因は未決定 |
| 色のかけら | 彩鎖からこぼれた魔力。ゲーム上では色ブロックとして表現する |
| マジカルチェイン | 同じ色の力を4個以上結び、乱れた魔力を解放・再接続する魔法 |
| 浄鎖 | 暴走した魔物を傷付けず、絡まった色の力をほどいて穏やかに戻すこと |

## クロマが旅をする理由

彩鎖魔法学院の見習い魔女クロマは、色を「結び直す」魔法に適性を持つ。各地の結び目を巡り、盤面へあふれた色のかけらをマジカルチェインで解放し、ほつれた彩鎖をつなぎ直すことが旅の目的である。

クロマは魔物を倒したり消滅させたりするのではない。魔物の中で絡まった色の力をチェインでほどき、元の穏やかな状態へ戻す。大きな連鎖ほど修復の力が高まり、地域の結び目も安定する。

## 魔物との対立

魔物は本質的な悪ではなく、彩鎖の乱れに巻き込まれた存在とする。

- モコスライムは、こぼれた色のかけらを抱え込みすぎて暴走した最初の魔物
- チェインバット以降も、それぞれの土地の彩鎖異常によって妨害能力が強くなっている
- 敗北時は消滅させず、浄化、縮小、改心、または穏やかになって退場する
- 将来は浄鎖した魔物がマップ、拠点、再戦画面へ再登場できる構成にする

## クロマとミラ

クロマとミラは彩鎖魔法学院の同級生で、互いを強く意識するライバルとする。

| クロマ | ミラ |
| --- | --- |
| 色をつなぎ、流れを整える | 鏡と鎖で対象を固定し、封じる |
| 状況へ合わせて柔軟に結び直す | 乱れを強くロックして制御しようとする |
| 寒色、プリズム、結びの鎖 | 暖色、割れ鏡、封印の鎖 |

ミラは完全な悪役ではない。自信満々に振る舞う一方でクロマへのライバル意識が強く、先回りして問題を解決しようとして失敗する「あわてんぼう魔女」である。強すぎる封印は彩鎖の流れまで止めてしまう場合があり、クロマとの勝負を通じて「鎖は縛るだけでなく、結び直すもの」だと理解していく。

ミラと戦う直接のきっかけは、ステージ構成と合わせて後で決定する。候補は次の3案を残す。

1. ミラが結び目を強制封印し、彩鎖の流れまで止めてしまう
2. クロマより先に異変を解決しようと、勝負を仕掛ける
3. 割れ鏡の魔力に影響され、本人の意思以上に封印魔法が暴走する

## ステージ進行の基本形

1. クロマが異常の起きた結び目へ到着する
2. 魔物またはミラによる妨害の兆候を確認する
3. 色ブロックをマジカルチェインで結び、乱れた魔力を解放する
4. 敵の妨害を越え、魔物を浄鎖する
5. 地域の彩鎖を再接続し、次の結び目へ向かう

## ゲームルールとの対応

| ゲーム上の要素 | 世界観上の意味 |
| --- | --- |
| 落下する色ブロック | 彩鎖からこぼれた色のかけら |
| 同色4個以上の消去 | マジカルチェインによる再接続 |
| 連鎖 | 複数のほつれを続けて修復した状態 |
| 特殊ブロック | クロマが一時的に結晶化した補助魔法 |
| 敵の妨害 | 魔物の暴走、またはミラの封印魔法 |
| ミラの封印ゲージ | ミラが固定しようとしている封印の強さ。0になると封印が解かれ、勝負が決着する |
| 魔物の乱れゲージ | 魔物に絡まった色の力の乱れ。0になると浄鎖が完了する |
| 敵撃破 | 浄鎖、鎮静、封印解除、勝負の決着 |
| ゲームオーバー | 色のかけらを処理しきれず、修復を一度中断した状態 |
