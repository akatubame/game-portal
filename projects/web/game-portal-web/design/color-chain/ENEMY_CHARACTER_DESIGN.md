- 世界観・物語構成: `WORLD_SETTING.md`

## 目的と共通方針

クロマの鎖魔法を「同じ色の魔力を結んで解放する力」とし、敵側は色や鎖の力が乱れたことで生まれた、浄化可能なゆるキャラ風魔物として描く。恐怖よりも表情の豊かさと可愛さを優先し、敗北時も救いのある反応を残す。

- 表現: クロマと同じアニメ調、明瞭な濃紺輪郭、セル塗り寄り、宝石色の発光
- 共通意匠: 小さな鎖輪、結晶、または色の乱れを各キャラクターへ1つ以上入れる
- 画面内の役割: クロマとは別の敵パネルに表示し、待機・予告・反撃・被弾・敗北を切り替える
- モバイル: 全身立ち絵ではなく、胸上またはデフォルメ版とHP表示を優先する
- 敗北: 消滅させず、浄化・改心・小さくなって退場する演出を基本とする

## 敵ステージの基本順

| 順番 | 敵 | 舞台案 | 役割 |
| --- | --- | --- | --- |
| 1 | モコスライム | 魔法の森 | 色魔法と敵の反応を学ぶ導入 |
| 2 | チェインバット | 月夜の洞窟 | 視界・横方向の妨害 |
| 3 | フロストン | 氷結の庭園 | 時間・操作の妨害 |
| 4 | ポンポンゴースト | 古書塔 | 情報の隠蔽・予測の妨害 |
| 5 | キノコノコ | 胞子の温室 | 特殊ブロックの見分けに関する妨害 |
| 6 | ミラ・ノワール | 黒鏡の塔 | 鎖魔法を使うボス戦 |

## 妨害設計の共通ルール

- 技は必ず「予告 → 発動 → 短時間で解除」の順にし、発動前に敵パネルと盤面へ分かる予告を出す
- 初期ステージでは、操作不能・色の完全非表示・連続妨害を使わない
- 妨害中でも最低限の盤面情報と次ブロックは確認できるようにする
- クロマのチェイン・特殊技で敵の妨害ゲージを削り、大技の発動を遅らせたり中断できる余地を設ける
- 難易度上昇は妨害の頻度より、予告時間の短縮・組み合わせ・敵HPの増加で調整する
- 将来の盤面回転式試作では、落下位置や最上段を前提にした妨害を、交点封鎖、短時間の視界妨害、領域制限へ置き換える。具体案と成立手保証は `ROTATION_PUZZLE_IMPLEMENTATION_PLAN.md` に従う

## キャラクター一覧

### 1. モコスライム / Moko Slime

色を食べて少し暴走している、最初に出会う迷子の魔物。倒すより浄化する印象を持たせる。

| 項目 | 設定 |
| --- | --- |
| シルエット | 横に少し広い、低めで丸い半透明ゼリー |
| 基調色 | ミント〜水色。内部にピンク・黄・青・紫の結晶片が漂う |
| 顔 | 大きな黒目、短い眉、小さな `ω` 型の口。怒っても怖くしない |
| 識別要素 | 額のひび入り小結晶、首元に切れた鎖輪を一つ |
| 被弾 | 体が平たく潰れ、内部結晶が星のように散る |
| 敗北 | 小さなスライムへ分裂し、結晶をクロマへ返して跳ねながら退場 |
| 基本妨害 | **ぬめりブロック**: 盤面下部へ1個だけ置く。周囲のブロックを一度だけ滑らせる軽い妨害 |
| 反撃 | ぷるんと大きく膨らみ、しずくと一緒にぬめりブロックを吐き出す |

モコスライムは視認性や操作を奪わない。敵との攻防を学ぶための導入キャラクターとする。

### 2. チェインバット / Chain Bat

月形の翼と鎖輪ネックレスを持つ、丸いコウモリ魔物。横方向へ動く演出を得意とする。

| 項目 | 設定 |
| --- | --- |
| シルエット | 真円に近い胴体、小さな牙、月形の翼 |
| 基調色 | 紺、紫、月光の淡いシアン |
| 識別要素 | 首の鎖輪、翼膜に小さな月と結晶の模様 |
| 基本妨害 | **ムーンミスト**: 盤面下2段を半透明の霧で短時間覆う。ブロックの輪郭と色の一部は見える |
| 大技候補 | **ムーンレス・フォグ**: 盤面下3段を2〜3秒だけ濃い霧で覆う。明確な予告と長めの再使用間隔を必須とする |
| 反撃 | 羽を広げて一回転し、鎖を振りながら霧を横方向へ流す |

下数段を完全に見えなくする妨害は強すぎるため、通常技には採用しない。完全隠蔽ではなく「半透明・短時間・予告あり」の大技として段階的に導入する。

### 3. フロストン / Froston

氷帽子と結晶の鼻を持つ、小型雪だるま魔物。冷気で盤面のテンポを乱す。

| 項目 | 設定 |
| --- | --- |
| シルエット | 二段雪玉、短い枝腕、少し大きい氷帽子 |
| 基調色 | 白、淡い水色、青紫、銀 |
| 基本妨害 | **フロスト・タップ**: 数マスを一時凍結し、操作の反応をわずかに重くする |
| 反撃 | 体を冷気で膨らませ、口から雪の息を吹く |

### 4. ポンポンゴースト / Pompon Ghost

丸い裾を持つシーツおばけ風の魔物。驚き・泣き・照れなどの顔芸を強みにする。

| 項目 | 設定 |
| --- | --- |
| シルエット | 大きな丸頭、波打つシーツ裾、宙に浮く短い腕 |
| 基調色 | 乳白色、薄紫、淡いピンク、金の小鈴 |
| 基本妨害 | **ゴースト・ミックス**: 次のブロックを短時間だけ `？` 表示にする |
| 反撃 | ぷくっと膨らんで盤面の前を横切り、煙を残して消える |

### 5. キノコノコ / Kinokonoko

大きな傘を帽子のように被った、小柄なキノコ魔物。胞子と色の偽装を使う。

| 項目 | 設定 |
| --- | --- |
| シルエット | 大きな傘、小さな丸胴体、短い手足 |
| 基調色 | 朱赤、紫、黄緑、胞子の虹色粒子 |
| 基本妨害 | **ミミック胞子**: 一定時間、特殊ブロックを通常色に見せかける |
| 反撃 | 傘を回し、胞子をぱっと散らす |

### 6. ミラ・ノワール / Mira Noir

クロマへ強いライバル意識を燃やす、負けず嫌いな「あわてんぼう魔女」。クロマの「結ぶ鎖」に対し、鎖と鏡で色を縛り、乱すお邪魔魔法を使う最初のボスである。自信満々に見せようとするが、想定外の大連鎖や被弾で慌てて素が出る。倒される敵ではなく、勝負を通じて関係を築くライバルとして描く。

- 正式名: ミラ・ノワール / Mira Noir
- ゲーム内の表示名・会話時の呼び名: ミラ / Mira
- 敵パネル、HP表示、技の発動者名、キャラクター選択では短い呼び名の `ミラ / Mira` を優先する
- 設定資料、ストーリー、初登場時の正式な紹介だけは `ミラ・ノワール / Mira Noir` を使用する

| 項目 | 設定 |
| --- | --- |
| 等身・印象 | クロマと近い小柄な等身。デフォルメ時は大きな帽子、丸い目、短いマントで可愛さを前面に出す |
| 衣装 | 明るい赤の魔女帽子、赤橙の短いマント、クリーム色と金の縁取り、朱色〜コーラルのワンピース。鎖飾りは控えめに金色で入れる |
| 髪・顔 | 赤茶〜栗色のセミロング。金の差し色。普段は得意げな笑みだが、焦ると眉と口が大きく動く表情豊かな顔 |
| 小物 | 先端に夕焼け色の割れ鏡片を持つ短い鎖杖。鏡片が浮遊して小さな丸い魔法陣を作る |
| 魔法色 | 朱赤、橙、金、コーラルピンク、暖かい白。クロマの寒色寄りの紺・シアン系イメージと対比させる |
| 性格演出 | お邪魔魔法の成功時は得意げにポーズを決めるが、失敗時や大連鎖を受けた時は帽子を押さえて慌てる |
| 被弾 | 鏡片が小さく弾かれ、驚いて杖を抱え込む。大ダメージでは帽子がずれ、あわてて直す |
| 敗北 | 鏡片が暖かく光を取り戻し、悔しそうにしつつ「次こそは」と立ち直る。完全な悪ではない余韻を残す |
| 基本技 | **ノワールチェイン**: 邪魔ブロックを少数、上部から落とす |
| 中妨害 | **ミラージュ・ロック**: 数個のブロックを鏡色に覆い、一定回数の消去まで色を見えにくくする |
| 大技 | **ブロークン・プリズン**: 盤面の一部を短時間だけ鎖で囲み、その範囲の移動または回転を制限する |
| 反撃 | 鏡を掲げて予告し、杖を大きく振り下ろして橙と金の鏡片魔法陣を展開する。決めポーズの直後に少しよろける仕草を入れる |

## 制作する資料

### 全キャラクター共通

1. コンセプト設定画: 正面寄りの全身、配色、識別小物、魔法エフェクトの方向性
2. 三面図: 正面・真横・背面。デフォルメ体型でも構造が分かるようにする
3. 表情シート: 通常、喜びまたは余裕、驚き、怒り、被弾、敗北
4. デフォルメ設定画: ゲーム内の敵パネル向け。胸上でも識別できる2.5〜3頭身
5. モーション原画: 待機、予告、反撃、被弾、敗北。各動作は開始・溜め・最大姿勢・戻りを基本とする
6. 小物・効果シート: 技固有の鎖、霧、氷、胞子、鏡片、邪魔ブロックなど

### モコスライムの優先制作順

1. コンセプト設定画
2. 三面図
3. 表情シート
4. 通常・反撃・被弾・敗北のモーション原画
5. ぬめりブロック、しずく、結晶片の小物シート
6. デフォルメ版とドット絵方向性

### ミラ・ノワールの優先制作順

1. コンセプト設定画
2. 三面図
3. 表情シート（得意げ、ライバル心、怒り、焦り、パニック、被弾、大ダメージ、悔しい敗北、立ち直り）
4. 通常等身の立ち絵とデフォルメ設定画
5. 詠唱・基本反撃・大技・被弾・敗北のモーション原画
6. 鎖杖、割れ鏡、魔法陣、邪魔ブロックの小物シート
7. 将来の第2形態を想定した変化案

## イラスト生成時の共通指定

- クロマと同じ世界のゲーム用アニメ調。明瞭な輪郭とセル塗り寄りの陰影にする
- 既存作品・既存キャラクターの固有の衣装や画風を指定しない
- 背景は白または淡い無地にし、資料画像には文字、ロゴ、透かしを入れない
- 一枚の資料では用途を混ぜず、三面図・表情・モーション・小物を分けて生成する
- 実装用素材へ進む前に、採用画像を決めて次の資料の参照画像として使う
- モーション素材は背景透過を見据え、輪郭の外側へ強い発光や煙を重ねすぎない

## 実装に進む前の確認

- 敵の妨害はステージ制を実装するまで、テスト表示・演出のみに留める
- クロマの操作性を阻害しすぎないよう、妨害の秒数・予告・解除条件を必ず個別に検証する
- 敵の表示ON/OFF、画面効果ON/OFF、ボイスON/OFFを独立して切り替えられる構造にする
- 初期実装では、敵HP・勝敗・報酬をゲームスコアへ影響させない
