# モコスライム 実装用ランタイム素材

モコスライムを対戦パネルと将来のステージ演出へ接続するための、背景透過PNG素材。
全素材は同一の 1254 x 1254 px キャンバスで、下端を基準に配置している。

## 配置先

`public/characters/moko/`

| 状態 | ファイル | 用途 |
| --- | --- | --- |
| 通常 | `moko-idle.png` | 対戦中の基本待機 |
| 反撃 | `moko-attack.png` | ぬめりブロックを放つ予告・反撃 |
| 軽被弾 | `moko-hit-light.png` | Wチェイン、軽被弾技 |
| 中被弾 | `moko-hit-medium.png` | トリプルチェイン、中被弾技 |
| 大被弾 | `moko-hit-heavy.png` | ハイパー以上、大被弾技 |
| 浄化 | `moko-purified.png` | 乱れゲージ 0% 到達時 |

## 共通仕様

- 背景は透過PNG。
- 体色、額の水色結晶、金色の切れた鎖、内部の4色結晶を共通の識別要素として維持する。
- 画像単体には文字・UI・画面全体のエフェクトを含めない。演出は実装側のCSS／Canvasで重ねる。
- ピンク結晶を保護するため、外側のクロマキー背景だけを判定して透過化している。
